# Neon Starwave

Jogo arcade de nave espacial com visual neon!

## Como jogar

- **Mover nave:** Use as setas do teclado ou WASD
- **Atirar:**
  - **Desktop:** Pressione a barra de espaço ou clique no botão "Atirar" no canto inferior direito
  - **Mobile:** Use o botão de tiro na tela
- **Pegue as bolas coloridas:** Elas aumentam a quantidade e a força dos tiros

## Controles

| Plataforma | Movimento | Tiro |
|------------|-----------|------|
| Desktop    | Setas/WASD| Espaço ou botão "Atirar" |
| Mobile     | Botões na tela | Botão de tiro |

## Como rodar

Abra o arquivo `main.html` em seu navegador favorito (Chrome, Firefox, Safari, etc).
