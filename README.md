# Neon Starwave

Jogo arcade de nave espacial com visual neon!

## Como jogar

- **Mover nave:**
  - **Desktop:** Use as setas do teclado ou WASD
  - **Mobile:** Use o D-pad (setas) na tela
- **Atirar:**
  - **Desktop:** Pressione a barra de espaço ou clique no botão "Atirar" no canto inferior direito
  - **Mobile:** Use o botão de tiro grande na tela
- **Pegue as bolas coloridas:** Elas aumentam a quantidade e a força dos tiros

## Controles

| Plataforma | Movimento           | Tiro                       |
|------------|--------------------|----------------------------|
| Desktop    | Setas/WASD         | Espaço ou botão "Atirar"   |
| Mobile     | D-pad na tela      | Botão de tiro grande       |

## Estrutura dos arquivos

- `index.html` — Apenas o HTML do jogo
- `style.css` — Todo o visual e responsividade
- `game.js` — Toda a lógica do jogo

## Como rodar

1. Baixe ou clone o projeto
2. Abra o arquivo `index.html` em seu navegador favorito (Chrome, Firefox, Safari, etc)
3. Para testar no celular, abra o arquivo no navegador do seu dispositivo ou use o modo mobile do navegador

## Dicas

- O jogo é responsivo: os controles mudam automaticamente para mobile ou desktop
- As bolas coloridas (power-ups) aumentam o poder dos tiros
- O botão "Atirar" aparece no desktop, e o D-pad + botão de tiro aparecem no mobile
